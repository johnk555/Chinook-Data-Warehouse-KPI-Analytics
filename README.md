📊 Chinook Data Warehouse & KPI Analytics Project

📌 Project Overview

This project demonstrates the design and implementation of a complete data warehousing and analytics solution using the Chinook Database.

The objective was to transform raw transactional data into a structured Data Warehouse and extract meaningful business KPIs using Power BI.

🎯 Business Problem

Transactional databases are optimized for daily operations, not analytics.

The goal of this project was to:

Extract analytical insights from operational data

Design a scalable Data Warehouse

Build automated KPI reporting

Enable business decision-making through dashboards
